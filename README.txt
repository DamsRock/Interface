Projet Interface Easy School
Réalisé par 	Antoine AOUAT

Utilisation de Python avec les modules suivants:
	sys
	os
	Tkinter/tkinter (python2/python3)
	pysftp
	requests
	lxml
	reportlab

Pour lancer correctement ce programme 
Il faut d'abord soit:
	• Un server sur lequel vous allez mettre les fichiers présents dans le dossier 'file_to' (voilà le chemin 'Interface\com\ecetech\b2\gui')
	de créer une VirtualHost avec pour ServerName www.mesdocumentsinterfaces.org
	et de ne pas oubliez d'ajouter dans le fichier hosts présent sur le PC depuis lequel vous souhaitez accéder en y ajoutant l'adresse ip du server et le www.mesdocumentsinterfaces.org
	(le fichier hosts se trouve ici 'C:\WINDOWS\system32\drivers\etc\' sur Windows ou là '/etc/' sous un OS Linux

	• De changer toute les requêtes où sont appelés les fichiers et de pointer vers les fichiers correspondants dans le dossier 'file_to'


Il faut lancer en premier le fichier 'log.py' (voilà le chemin: 'Interface\com\ecetech\b2\gui')
Les autres fichiers qui sont appelés se trouve dans le dossier interface_eleve (chemin: 'Interface\com\ecetech\b2\gui\interface_eleves'


Dans le dossiers Docs se trouve plusieurs choses : 
	• Des aides qui nous ont permis de mener à bien se projet;
	• La vidéo de l'interface Easy School qui sert de démontration et pour ceux qui ne pourrait la faire fonctionner chez eux (cela donne un avant goût du travail effectuer)
